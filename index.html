Web Designerを目指したきっかけ

昔から、かぎ針編みやレトロフォントの写真集めなど、 “惹かれたものを形にする”のが大好きでした。

ロジスティックコーディネーターとして働く中で、 複雑な情報を整理して伝える力が身につき、

「これってデザインにも活かせるかも」

と気づいたのが転機です。

カナダで暮らしながら出会った色づかいやレイアウトの自由さに惹かれ、 自分の世界観を Web で表現したいと思い、

Web デザイナーを目指しています。

新しい技術もどんどん吸収していきます。
